КАК ОТКРЫТЬ НА iPhone И СДЕЛАТЬ КАК ПРИЛОЖЕНИЕ (БЕСПЛАТНО)
==========================================================

ВАЖНО: офлайн на iPhone работает после первого открытия сайта онлайн (он кэшируется).

1) Запуск на ПК (для проверки)
------------------------------
- Установи Node.js (LTS)
- Распакуй проект
- В терминале внутри папки проекта:

  npm install
  npm run dev

- Открой: http://localhost:5173

2) Сделать доступным на iPhone (чтобы добавить на экран)
-------------------------------------------------------
Нужно выложить сайт бесплатно. Самый простой вариант — Cloudflare Pages.

Шаги:
A) Собери сайт:
   npm run build
   -> появится папка dist

B) Залей на Cloudflare Pages (free):
   - Создай GitHub репозиторий
   - Загрузи туда проект (весь)
   - В Cloudflare Pages подключи репозиторий
   - Build command: npm run build
   - Output folder: dist

C) На iPhone:
   - Открой сайт в Safari
   - Поделиться -> "На экран Домой"
   - Открой уже с иконки

3) Офлайн
---------
- Первый запуск должен быть онлайн (чтобы кэш скачался).
- Потом можешь включать авиарежим — будет открываться.

4) Синк через iCloud
--------------------
Вкладка "Бэкап":
- Экспорт -> сохранить файл в iCloud Drive
- Импорт -> взять файл из iCloud Drive на другом устройстве

Файл зашифрован твоим паролем.


НОВОЕ: вкладка «Ранги»
- серия (streak)
- уровни и очки
- квесты недели
- достижения
(это офлайн-версия геймификации — без соцсетей и лидербордов)
